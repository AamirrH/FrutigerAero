# 🖥️ Frutiger Aero – Windows 7 Nostalgia in the Browser

**Frutiger Aero** is a fully interactive, browser-based recreation of the classic **Windows 7 “My Computer” experience**, designed to revive the iconic **Frutiger Aero aesthetic** of the late 2000s.

This project focuses on **visual accuracy, smooth interactions, and pure frontend craftsmanship**, recreating the glassy UI, soft gradients, retro folder naming, nostalgic sounds, and window behaviors — all **without using any frameworks**.

---

## ✨ Project Inspiration

Frutiger Aero represents a design era defined by:

* Glass-like transparency (Aero Glass)
* Soft gradients and glowing highlights
* Clean typography and skeuomorphic UI elements
* Glossy icons, calm wallpapers, and cozy desktop nostalgia
* That old Windows 7 / early 2010s feeling of just exploring your computer

This project aims to **preserve and relive that design philosophy** in a modern browser environment.

---

## 🚀 Features

* 🪟 **Interactive Window System**

  * Draggable window-style panels
  * Minimize, restore, and close button behavior
  * Classic desktop-style interactions

* 🗂️ **Classic “My Computer” Layout**

  * Retro folder and drive names
  * Familiar Windows 7-style structure
  * Pixel-aligned spacing and icons

* 🎨 **Authentic Frutiger Aero Styling**

  * Glassy panels and soft shadows
  * Gradients inspired by Aero Glass
  * Windows 7-era color palette
  * Glossy desktop icons and nostalgic visual assets

* 🖱️ **Mouse-Driven Interactions**

  * Click-to-open folders and pages
  * Hover effects that mimic native UI
  * Smooth transitions for realism

* 🔊 **Nostalgic Audio Experience**

  * Windows-style startup sound
  * Background music for the desktop atmosphere
  * Error sound effects for the full retro computer vibe

* 🖼️ **Picture Gallery**

  * Browse nostalgic images
  * Gallery-style interface inspired by classic desktop media viewers
  * Ambient audio for a softer Frutiger Aero mood

* 🌐 **Internet Explorer-Inspired Page**

  * Retro browser window layout
  * Old-school web browsing feel
  * A tribute to the classic internet era

* 🎮 **Games Section**

  * Desktop games page for that “old PC distraction” feeling
  * Styled to fit the Windows 7-inspired environment

* 📝 **Notepad Page**

  * Simple editor-style layout
  * Inspired by the familiar Windows Notepad experience

* 💙 **BSOD Throwback**

  * Blue screen-style page
  * Classic Windows crash nostalgia
  * Dramatic error-page energy

* ⚡ **Pure Frontend Implementation**

  * No frameworks
  * No libraries
  * Just **HTML, CSS, and vanilla JavaScript**

* 🚀 **Vercel Deployment Support**

  * Static site deployment
  * Custom rewrites for the restructured `/html` folder
  * Works cleanly from the deployed root URL

---

## 🧠 Tech Stack

* **HTML5** – semantic structure and layout
* **CSS3** – gradients, shadows, transparency, animations
* **Vanilla JavaScript** – window logic, interactions, audio, and page transitions
* **Vercel** – static deployment

> No React. No Vue. No Tailwind.
> This project demonstrates **core frontend fundamentals**.
